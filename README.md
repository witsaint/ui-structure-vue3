# UI Components
UI Components for everyone

## Introduction

```bash
.
├── LICENSE
├── README.md
├── vite.demo.config.js (demo网页vite配置)
├── vite.md.config.js (doc文档vite配置)
├── dist (文档编译目录)
├── examples (文档源文件)
│   ├── docs (多语种Markdown文档)
├── lib (库文件编译目录)
├── node_modules
├── package.json
├── UI (库项目源文件)
│   ├── components
│   ├── index.js
├── static (仅存放不需要处理的资源)
├── test (暂无)
│   └── unit
├── md.js (解析md)
└── package.lock
```

## Install

添加Dependencies：

```json
npm i gao-ui
```

```bash
yarn
```

## Quick Start

```bash
yarn dev
```
